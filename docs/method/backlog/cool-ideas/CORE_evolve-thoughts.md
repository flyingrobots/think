# Evolve a thought with explicit lineage

Start a new thought seeded from an old one.

```
think --evolve=<entryId> "actually, the real insight is..."
```

Like capture but with a `seeded_by` edge back to the original.
Browse shows the evolution chain. A raw seed becomes a developed
idea through multiple passes — each pass is a new immutable entry.

Different from reflect (which pressure-tests) and annotate (which
comments). Evolve says "this thought grew into this new thought."
