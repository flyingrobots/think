# Link thoughts explicitly

Connect two thoughts with an explicit relationship.

```
think --link <id1> <id2> "related because..."
```

A `relates_to` edge with an optional description stored as an edge
property. Browse shows linked thoughts in a "Connections" panel.
Builds the graph from a flat chronology into a navigable web.

Could also support typed links: "contradicts", "extends", "replaces",
"inspired_by".
