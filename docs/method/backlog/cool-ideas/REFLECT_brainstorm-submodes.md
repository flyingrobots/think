# Brainstorm submodes and constraint injection

## Submodes

Possible brainstorm flavors: diverge, refine, challenge, recombine. These should only appear if they make brainstorm sharper rather than more menu-driven.

## Constraint injection

In brainstorm mode, inject a deliberate constraint to sharpen thinking. Examples: "What if you had 10 minutes to implement this?", "What if this had to work offline?", "What if this had to be explainable in one sentence?"
